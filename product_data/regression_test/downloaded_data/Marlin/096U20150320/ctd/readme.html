Data Quality Flags

Each row in the extracted CSV file contains data quality flags. It is the column to the immediate right of that particular parameter suffixed with '_QC'. A complete listing of these codes is in Data Download format document.

The DQ flags used in this file are.

  • 32 - No error - data is good (State: Good. Operation: Filtered)
  • 44 - Unprocessed (uncalibrated) or processing error (State: Good. Operation: Filtered)
  • 141 - No data - data missing for unknown reason (State: Bad. Operation: None)
