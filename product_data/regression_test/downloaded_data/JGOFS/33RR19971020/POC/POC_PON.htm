POC_PON 

 PI:              Walker O. Smith
 of:              University of Tennesee, Knoxville
 dataset:         POC and PON from bottle casts
 project/cruise:  AESOPS/KIWI06, APFZ Survey cruise 1

 Methodology: Please see Chapter 15 of the JGOFS protocols (1994),
 "Determination of Particulate Organic Carbon and Particulate Orgranic Nitrogen"

 PI-Notes on Methodology: 

 Parameter   Description                             Units


 event       event number from event log
 sta         station number from event log
 cast        cast number from event log
 cast_type   CTD = CTD rosette
             TM = trace metal rosette
 bot         rosette bottle number
 depth_n     sample depth                            meters
 POC_orig    particulate organic carbon              micrograms/liter
              in units received from P.I.
 POC         particulate organic carbon              micromoles/liter
              units converted by US JGOFS DMO
 PON_orig    particulate organic nitrogen            micrograms/liter
              in units received from P.I.
 PON         particulate organic nitrogen            micromoles/liter
              units converted by US JGOFS DMO
 nd          no data / bad data
